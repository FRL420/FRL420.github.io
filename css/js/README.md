# 📋 待办事项应用（To-Do List App）

一个功能完整的待办事项应用，支持本地存储、分类、优先级管理等功能。

## ✨ 功能特性

- ✅ **添加/删除任务** - 轻松管理你的日常任务
- 🎯 **任务分类** - 工作、生活、学习、其他
- ⭐ **优先级设置** - 高、中、低三个等级
- 🔍 **搜索过滤** - 快速找到你需要的任务
- 📊 **统计信息** - 显示总数、完成数、完成率
- 💾 **本地存储** - 数据自动保存到浏览器 localStorage
- 📱 **响应式设计** - 在手机、平板、电脑上都能使用
- ✓ **标记完成** - 点击勾选完成状态
